# Moontower Generator

## Description

A shell script to generate arbitrarily long ASCII-art moontowers

Usage: call the script with the number of segments you want to be in the final moontower.

Example:

```
username@moontower:~/mt-gen$ ./mt-gen.sh 2
    O\ /O
/-----+-----\
O / / | \ \ O
  |O /|\ O|
  | / | \ |
  \/  |  \/
   ---+---
  | _-|-_ |
  |/__|__\|
  |   |   |
  | _-|-_ |
  |/__|__\|
  |   |   |
  |-------|
   \ | | /
    \| |/
     | |
     | |
     | |
     | |
```
